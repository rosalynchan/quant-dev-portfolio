# Trading Data Multi-Agent Assistant

Multi-agent system for automated trading data ingestion, quality assurance, and reporting.

## Overview

Automates what traders do manually: load data, check quality, fix issues, generate reports.

Uses specialized AI agents that work together:
- **IngestionAgent**: Load data from CSV, Parquet, Excel
- **CleaningAgent**: Remove duplicates, handle missing values, detect anomalies
- **AnalysisAgent**: Calculate VWAP, OHLC, spreads, statistics
- **ReportingAgent**: Generate HTML reports with metrics
- **PlannerAgent**: Orchestrate the workflow

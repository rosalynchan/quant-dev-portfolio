# Quant Etl Pipeline

## Project Overview
ETL pipeline for ingesting, transforming, and loading large financial datasets.
用于处理大型金融数据的ETL流水线，支持提取、转换与加载。

## Tech Stack
- Python
- Pandas, NumPy, FastAPI / Dash (as needed)
- Docker, GitHub Actions
